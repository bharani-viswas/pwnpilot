# Quality evaluation tests — Phase F

This package contains repeatable quality-benchmark tests aligned to practical
run-quality metrics from recent agent pentesting literature.

Gates tested:
  - Loop-rate / repetition controls
  - Operator visibility and intervention counting
  - Report finalization (closure reliability)
  - Replay completeness
  - Memory recall quality (finding retrieval)
  - Approval audit integrity
  - No-silent-stop behavior
