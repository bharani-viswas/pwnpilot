# Rules of Engagement - PwnPilot Demo

## Engagement Details
- **Target**: OWASP Juice Shop (Demo Vulnerable Application)
- **Scope**: http://localhost:3000
- **Type**: Security Assessment & Vulnerability Testing
- **Authorization**: Demo/Educational Purpose Only

## Authorized Activities
1. **Reconnaissance**
   - Web application scanning
   - Directory/file enumeration
   - API endpoint discovery
   - Technology/framework enumeration

2. **Vulnerability Assessment**
   - SQL injection testing
   - Cross-site scripting (XSS)
   - Cross-site request forgery (CSRF)
   - Authentication/authorization flaws
   - Sensitive data exposure
   - Broken business logic
   - XML/XXE injection

3. **Exploitation (as found)**
   - Proof-of-concept exploitation
   - Impact demonstration

## Restrictions
- No Denial of Service (DoS) attacks
- No resource exhaustion
- No permanent data destruction
- No unauthorized access to system resources
- No social engineering
- Testing hours: 24-hour window from engagement start

## Authorization
This engagement is authorized for educational and demonstration purposes only.
